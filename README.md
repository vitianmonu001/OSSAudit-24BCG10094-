# Chosen Software

In this project, I have worked on Linux and Git. Linux is used as the main operating system, and Git is used for version control.

# About the Project

This project is mainly about understanding how open-source software works in a real system. Instead of only studying theory, I tried to see how things actually work in a Linux environment.

Linux is widely used in servers and development, and Git is commonly used to manage code and track changes. By working with both, I got a basic idea of how developers use these tools in real-world situations

# Script 1 – System Identity Report

This script shows basic details of the system like user name, kernel version, uptime, and OS.

# Script 2 – Package Inspector

This script checks whether a package like Git  is installed and shows its version.

# Script 3 – Disk and Permission Auditor

This script checks some important directories and displays their permissions and size.

# Script 4 – Log File Analyzer

This script reads a log file and counts how many times a keyword like "error" appears.

# Script 5 – Manifesto Generator

This script asks some questions and creates a small text file based on the answers.

# How to Run

First give permission:
chmod +x script.sh

Then run:
./script.sh

For log analyzer:
./script4.sh /var/log/syslog
